# Ceyline Land

A 2D portfolio disguised as a game.

Explore Ceyline's Island and her house. If you're hiring, please end her unemployment arc.

WASD or click/tap to move.
Turn on volume for immersion.

---

## How to run

You need Node.js and npm. Then:

```bash
npm install
npm run dev
```

Open the URL it prints.

---

## How to build

```bash
npm run build
```

A `dist` folder will appear.

---

## How to preview the build

```bash
npm run preview
```

---

## Who and what you will find

### On the island

- **Welcome Bunny** – Your first encounter. Welcomes you to Ceyline's Island.
- **Boyfriend** – Ceyline's boyfriend. Wants her hired so he can upgrade from tent to terrace.
- **Guard** – Guards the castle. The trust fund is frozen; she has to grind for her own gold.

### Inside the house

- **Sister** – Ceyline's older sister. Retiring from software to open a bakery. Please hire Ceyline to fund the bakery era.
- **Ghost** – Appears when Ceyline procrastinates. Whispers "deadline."
- **Computer** – Ceyline's coding station. GitHub: laggylogic.
- **Resume** – On the desk. Open before it updates itself.
- **Fridge, matcha station, certificate, study table** – Click around to explore.
- **Book shelves, bed, hobby station** – More of her world.
